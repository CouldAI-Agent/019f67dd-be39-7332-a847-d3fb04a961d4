# MediChannel App

A companion mobile application for a medical and medicine educational YouTube channel.

## Overview
MediChannel provides a centralized hub for medical students, healthcare professionals, and enthusiasts to watch informative videos, read up on clinical topics, and stay updated with the latest from the YouTube channel.

## Features
- **Home Dashboard:** Quick access to latest video uploads and highlighted medical topics.
- **Medical Topics Hub:** Categorized learning modules spanning Anatomy, Pharmacology, Pathology, and more.
- **Video Library:** A comprehensive listing of all channel lectures and tutorials.
- **About:** Channel details and a quick link to subscribe on YouTube.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart

## Setup
1. Ensure you have Flutter installed (version 3.7.2 or higher).
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app on your preferred emulator or device.
